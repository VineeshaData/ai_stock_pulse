AI Stock Pulse
An automated AI-driven stock analysis tool that tracks news sentiment and generates investment reports.

Features
Sentiment Analysis: Analyzes recent news to determine market mood.

Investment Reports: Generates CSV reports like final_investment_report.csv.

Interactive Dashboard: Visualizes data using Streamlit.

Project Structure
dashboard.py: The main UI for the project.

ai_sentiment_report.py: Logic for analyzing news headlines.

check_news.py: Script to fetch the latest market updates.

How to Run
Install dependencies: pip install pandas streamlit

Run the dashboard: streamlit run dashboard.py
